WolfPack Volleyball – Touch Logger PWA

Usage:
1. Upload these files to GitHub Pages (root of repo).
2. Open the GitHub Pages URL in Safari (iPhone/iPad).
3. Tap Share → Add to Home Screen.
4. The app will install with the WolfPack icon and work offline.

Files included:
- index.html
- manifest.json
- service-worker.js
- /assets (icons)
